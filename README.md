## 📌 Project Overview
The **Electric Cars Dashboard** is a data visualization project built using **Tableau** to analyze and explore trends in electric vehicles.  
This dashboard provides insights into electric car performance, including battery capacity, driving range, price comparison, and brand analysis.

The goal of this project is to present electric vehicle data in a **clear, interactive, and visually appealing way**.

---

## 🎯 Project Objectives
- Analyze electric vehicle data
- Compare different electric car brands
- Visualize battery capacity and range
- Understand price vs performance trends
- Build an interactive dashboard using Tableau

---

## 🛠️ Tools & Technologies
- **Tableau Public / Tableau Desktop** – Data Visualization
- **MySQL** – Database Management
- **CSV Dataset** – Data Source
- **Excel** – Data Cleaning (optional)

---

## 📊 Dashboard Features
The dashboard includes multiple visualizations such as:

- 🔋 **Battery Capacity Analysis**
- 🚗 **Electric Car Brand Comparison**
- 💰 **Price vs Range Analysis**
- 📈 Performance Trends
- 🎛 Interactive Filters and Charts

These features allow users to easily explore electric vehicle data.

---

## 📂 Dataset Information
The dataset contains details about electric cars including:

- Brand
- Model
- Battery Capacity (kWh)
- Range (km)
- Price
- Charging Time
- Top Speed

The dataset was imported into **Tableau using CSV and MySQL connection**.

## ⚙️ Project Workflow
